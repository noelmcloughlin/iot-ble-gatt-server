# iot-edge-stepping-stones

Collection of IoT solutions and ideas (MQTT, GATT, BLE, BT, PI, SENSORS, PLATFORMS) for connected devices.

Technologies
=============
* BLE advertising
* MQTT Broker

Edge platforms
==============
* EdgeXfoundry

IoT Clouds
===========
* WiA
* ThingSpeak
* Blynk

Boards
======
* BME680 module
* Sensornode MCU
* Pi-SenseHat
* Raspberry Pi
