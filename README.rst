# iot-pi-stepping-stones

Collection of IoT solutions and ideas (MQTT, GATT, BLE, BT, PI, SENSORS, IOT PLATFORMS) for Pi and other devices.

Contents
========

* BLE advertising
* MQTT Broker
* SenseHat sensors
* Edge platforms: EdgeXfoundry
* IoT platforms: WiA, ThingSpeak, Blynk.
